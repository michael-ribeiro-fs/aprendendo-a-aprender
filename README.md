# Aprendendo a Aprender

Projeto desenvolvido como parte da formação em **Web Development**, com foco na **prática consciente e aplicação real** dos conceitos estudados até o momento.

O objetivo principal não foi apenas “fazer funcionar”, mas **construir uma página completa, organizada e tecnicamente correta**, seguindo boas práticas usadas no mercado.

---

## 📌 Objetivo do Projeto

Este projeto teve como finalidade **consolidar os fundamentos de front-end** por meio da construção de uma landing page informativa sobre **técnicas de aprendizagem** e **curiosidades sobre o processo de aprender**.

Todo o desenvolvimento foi orientado à prática de:

- Escrita de **HTML semântico**
- Organização de estilos com **SCSS**
- Uso consistente da **metodologia BEM**
- Layouts com **Flexbox**
- **Posicionamento semântico e visual**
- Código limpo, sem seletores mortos ou redundantes
- Uso correto de **Git e GitHub** no fluxo de desenvolvimento

⏱️ **Tempo total de desenvolvimento:** aproximadamente **11 horas**

---

## 🛠️ Tecnologias e Conceitos Utilizados

- **HTML5**
  - Estrutura semântica (`header`, `main`, `section`, `footer`)
  - Hierarquia correta de títulos
  - Boas práticas de acessibilidade básica (`alt`, organização do conteúdo)

- **SCSS**
  - Aninhamento para organização visual
  - Estrutura modular por blocos
  - Código validado manualmente, sem erros de sintaxe

- **CSS / Layout**
  - Flexbox para alinhamento e distribuição
  - Controle de posicionamento (`relative`, `absolute`, `z-index`)
  - Layout de página completa (full viewport sections)

- **Metodologia BEM**
  - Nomeação consistente (`block__element--modifier`)
  - Clareza na leitura e manutenção do código
  - Evita colisões e ambiguidade de estilos

- **Git & GitHub**
  - Versionamento do código
  - Organização do projeto
  - Histórico de desenvolvimento
